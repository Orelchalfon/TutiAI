# RaayentiAI

**RaayentiAI** is an AI-powered tutoring platform designed to assist learners across multiple subjects. Built using TypeScript, this project leverages modern AI technologies to deliver personalized educational experiences.

## Features

- Multi-subject AI tutoring
- Personalized learning paths
- Interactive Q&A sessions
- Progress tracking and analytics

## Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Orelchalfon/RaayentiAI.git
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the application:**
   ```bash
   npm run start
   ```
